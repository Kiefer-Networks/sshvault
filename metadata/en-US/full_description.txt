SSHVault is a cross-platform SSH and SFTP client with zero-knowledge encrypted cloud sync. All credentials, SSH keys, and server configurations are encrypted client-side with AES-256-GCM before leaving your device. The server never sees plaintext data.

Features:
- SSH terminal with xterm emulation and 256 colors
- Split view, tabs, and multiple simultaneous sessions
- SFTP file browser with bookmarks, inline editor, and archive extraction
- Host key verification (Trust On First Use) with known hosts management
- SSH key management (Ed25519, RSA, ECDSA generation and import)
- SSH config import from ~/.ssh/config
- Jump host support (ProxyJump for multi-hop connections)
- Snippet manager for saved commands
- Post-connect commands for automation
- Server organization with folders, tags, color codes, and search
- Proxy support (SOCKS5 and HTTP)
- Biometric lock with PIN fallback and duress PIN
- Self-hosted sync server (Docker)
- End-to-end encrypted sync across unlimited devices
- Export and import of all data
- 27 languages supported
- Material 3 design on all platforms

Security:
- Zero-knowledge architecture: the server cannot read your data
- AES-256-GCM encryption with Argon2id key derivation
- Counter-based nonces to prevent nonce reuse
- DNS-over-HTTPS with multi-resolver cross-check
- Server attestation via Ed25519 TOFU key pinning
- No tracking, no analytics, no telemetry
- Fully open-source backend (AGPL-3.0)

The self-hosted sync server is open source: https://github.com/Kiefer-Networks/sshvault-api

SSHVault is completely free with no in-app purchases, no subscriptions, and no feature restrictions. Self-hosting the sync server is free and requires only Docker and PostgreSQL.
